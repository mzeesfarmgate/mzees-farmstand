# Project Overview
PHP frontend web application.

## Tech Stack
- PHP
- HTML/CSS/JavaScript (frontend)
- Git + GitHub for version control

## Common Commands
- Start local server: `php -S localhost:8000`

## Coding Standards
- Follow PSR-12 coding standards for PHP
- Use meaningful variable and function names
- Add comments for complex logic

## Git Workflow
- Always create a new branch for features or fixes
- Write clear, descriptive commit messages
- Never commit directly to main/master

## Rules
- NEVER read, modify, or reference any .env files or files containing secrets/credentials
- NEVER commit sensitive information to Git
- Always ask before running destructive commands (deleting files, dropping databases, etc.)
- Keep changes focused — one task at a time
